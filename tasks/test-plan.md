# Math Teacher アプリ テスト計画

> 対象フェーズ: Phase 1–6（RAG/エージェント/UX含む）

---

## 0. 前提条件チェック

| # | 項目 | 確認方法 |
|---|------|---------|
| 0-1 | `OPENAI_API_KEY` が設定されている | `echo $OPENAI_API_KEY` |
| 0-2 | poppler がインストール済み（`pdftoppm`） | `which pdftoppm` |
| 0-3 | Python sidecar が動作する（埋め込みモデル） | `python3 sidecar/embed.py --help` |
| 0-4 | アプリが起動する | `cargo tauri dev` でエラーなし |

---

## 1. PDF 取り込み（Ingest）

| # | テスト内容 | 期待結果 |
|---|-----------|---------|
| 1-1 | 「PDFを開く」でファイル選択 | ファイルパスがヘッダに反映される |
| 1-2 | 「サンプルPDF」ボタン | `text_linear_algebra.pdf` が読み込まれる |
| 1-3 | 「LaTeX取り込み」実行 | ヘッダにページ進捗（`ingest-progress` イベント）が表示される |
| 1-4 | 取り込み完了後 | 「索引 N」が 0 より大きくなる |
| 1-5 | 「取り込み停止」ボタン | 進行中に停止でき、ハングしない |
| 1-6 | 同じPDFを再度取り込み | クラッシュせず、ページが上書き更新される |
| 1-7 | PDF未選択で「LaTeX取り込み」 | ボタンが disabled のまま |
